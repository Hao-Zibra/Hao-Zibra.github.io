Physics-based battery aging simulation with PyBaMM

An open-source PyBaMM Doyle–Fuller–Newman (DFN) pipeline that injects coupled aging — SEI growth, lithium-inventory loss, active-material loss, and particle cracking — then simulates 1000-cycle capacity fade and the evolution of impedance and DRT features. It reproduces a capacity knee and a matching polarization-resistance acceleration, and shows — at the pattern level — that the DRT polarization-resistance growth is a degradation signature shared with two measured public datasets.

1 · The model and its baseline

The workhorse is the DFN electrochemical model — it resolves solid-phase diffusion and the deep-DOD behavior where fade physics lives. A baseline check confirms the model choice before any aging is layered on: the reduced-order SPMe model diverges from DFN by up to ~320 mV over the 75–99% DOD window, so the full DFN carries all the cycling here.

2 · A thousand cycles of coupled aging

Three configurations run to 1000 cycles: the full O'Kane 2022 degradation recipe, a knee-forcing config (particle cracking ×25, softened from the literature ×50/×30 so it produces a clear knee and still survives all 1000 cycles), and a no-degradation control. A change-point detector locates the capacity knee where fade accelerates 2.16×.

NMC PUBLIC-PARAM SIM Thousand-cycle capacity retention. Three configs vs cycle number; the dotted line marks the detected knee (cycle 677) in the cracking config. End retention: 100.0% control, 96.8% base, 94.3% knee.
NMC PUBLIC-PARAM SIM Mechanism decomposition. Loss split into lithium-inventory (LLI) and negative active-material (LAM) loss. In the knee config (dashed) the knee is accelerating LLI — LAM is disabled in this ablation, isolating the mechanism.

All gates pass: monotonicity, closure, knee detection, and control drift (control stays flat at exactly 100.0%). Per-segment subprocess isolation keeps peak memory in budget.

3 · The impedance analogue: EIS → canonical DRT

At every ~100 cycles the aged cell is probed with a linearized EIS sweep (10 kHz–1 mHz) and deconvolved with the canonical distribution-of-relaxation-times engine. The knee that shows up in capacity has an impedance twin: the polarization resistance R_pol accelerates in exactly the config that knees. This cell's aging impedance is an ohmic shift plus polarization/diffusion growth — not a charge-transfer semicircle — so R_pol (total polarization) is the tracked interfacial quantity.

NMC PUBLIC-PARAM SIM Polarization-resistance evolution. R_pol vs cycle. The knee config accelerates (7.6 → 17.1 mΩ, 2.25×) while the base config stays gentle and the control is flat — the impedance analogue of the capacity knee.

4 · Pattern-level comparison with public measured data

The same DRT engine is run on the simulated cell and on two measured public datasets, so the R_pol-growth signature is computed identically on both sides. The question is deliberately modest: does the type of degradation signature transfer across chemistries? The answer is a qualified yes — read strictly at the pattern level.

5 · Honest limitations

  • Liquid NMC model, not a solid-state cell; all cross-dataset correspondence is pattern-level.
  • The knee accelerant (cracking multiplier) is a literature device to force a knee within feasible compute; its magnitude was chosen for survival to 1000 cycles, not fit to data.
  • The model has no distinct charge-transfer arc; a closer analogue to a measured interfacial resistance would need an explicit film/contact-resistance addition (future work).
  • The simulated knee cell ends at ~94% SOH (never reaches 80% end-of-life), so its knee-location fraction is relative to the run window, not a true cycle-life.
Scope. This page displays figures and plain conclusions from a public-input methods pipeline. The pipeline source is not released here — it imports a private shared DRT/figure engine — so a public code release would go through a separately sanitized package, not this repository. What this page demonstrates is physics-based-modeling discipline: pick the right-fidelity model, inject coupled degradation, validate the numerics, and compare to measured data only as far as the chemistry honestly allows.
